# Mood

Journal app for tracking mood.

## Features:

- [Clerk Autherntication](https://clerk.com/)
- User can add journal entries
- Mood prediction (and summary) Using OpenAI API
- Ask Questions to AI
- A chart for tracking mood and avg sentiment score
